# Ejecutar en desarrollo

1. Clonar repositorio

```
git clone url repository
```

2. Install dependencies

```
npm install
```

3. Build the database

```
docker-compose up -d
```

3. Run the project

```
npm run dev:start
```

4. Run the Unit Tests

```
npm run test
```
